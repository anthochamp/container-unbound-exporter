# Unbound Exporter Container

Container images based on [unbound_exporter](https://github.com/letsencrypt/unbound_exporter) (Let's Encrypt), exposing Unbound DNS resolver statistics as Prometheus metrics.

Sources are available on [GitHub](https://github.com/anthochamp/container-unbound-exporter).

See [README.md](README.md) for full documentation.

## Image tags

- `x.y.z`: Container image version `x.y.z`.
- `edge`: Latest commit build.

**Tag aliases:**

- `x.y`: Latest patch of `x.y`.
- `x`: Latest minor+patch of `x`.
- `latest`: Latest `x.y.z` tag.
